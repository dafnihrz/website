HTML Terms

HTML
(HyperText Markup Language) is a markup language to tells web browsers how to display a web page's contents.
HTML element
Formats the document's content.
HTML tag
Contains the HTML Element.
Angle Brackets
<   > enclose tags.
HTML attribute
Special words inside the tag that controls the element's behavior.
HTML object element
Inserts objects instead of enclosing text.
HTML special character
Represents characters that would be considered code in HTML.
